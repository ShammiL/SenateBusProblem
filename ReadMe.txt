#Running the program

1. Open a command line and navigate to the location of the Java executable, SenateBusProblem.jar.
2. Type the command,
   ```
   java -jar SenateBusProblem.jar
   ```
   and hit enter to start the program.

#Terminating the program

1. Type either 'Q' or 'q' on the command line and hit enter to exit the program.

#Source files

1. The relevant source files can be found inside the folder 'source_files'.
